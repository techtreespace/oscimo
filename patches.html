HOW TO USE PATCHES

패치 사용 방법

01
plugdata에서 패치 설계

plugdata(Pure Data)에서 신디사이저 구조를 설계합니다. 완성된 패치를 .pd 파일로 저장하세요.

저장 파일 형식: .pd
02
.uf2 로 변환

아래 컴파일러 도구에 .pd 파일을 업로드하고 컴파일 버튼을 누르면 firmware.uf2 파일로 다운로드됩니다.

출력 파일: firmware.uf2
03
oscimo에 업로드

oscimo를 BOOTSEL 모드로 연결하면 USB 드라이브로 인식됩니다. firmware.uf2 파일을 드래그&드롭하면 자동으로 설치됩니다.

드래그&드롭으로 즉시 설치
임시 컴파일러 · 운영 중
PD → UF2 컴파일러

.pd 파일을 업로드하고 Upload & Compile 버튼을 누르면 oscimo에 업로드 가능한 firmware.uf2 파일을 다운로드할 수 있습니다.

http://130.110.9.11:5000
컴파일러 열기 →
⚠ 현재 컴파일러는 임시 운영 중입니다. 추후 oscimo 전용 컴파일 툴로 교체될 예정입니다.
oscimo 업로드 가이드

oscimo에 패치 올리는 법

1
BOOTSEL 모드로 연결

oscimo의 BOOTSEL 버튼을 누른 상태에서 USB를 연결합니다. USB 드라이브(RPI-RP2)로 인식됩니다.

2
UF2 파일 복사

컴파일러에서 다운로드한 firmware.uf2 파일을 RPI-RP2 드라이브로 드래그&드롭합니다.

firmware.uf2 → RPI-RP2/
3
자동 재시작

파일 복사가 완료되면 oscimo가 자동으로 재시작되며 새 패치가 바로 실행됩니다.

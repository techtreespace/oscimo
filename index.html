plugdata.org — 무료 설치

PLUGDATA

소프트웨어로
소리를 설계한다

plugdata는 Pure Data 기반의 무료 비주얼 프로그래밍 툴입니다.
블록을 연결하듯 신디사이저 회로를 설계하고, 그대로 oscimo에서 실행합니다.

oscimo-patch.pd — plugdata
🔗
비주얼 패치 프로그래밍

코딩 없이 블록을 연결해 신디사이저 회로를 설계합니다. VCO, VCF, VCA, LFO, ADSR — 원하는 구조를 자유롭게 조립하세요.

📤
oscimo로 직접 업로드

설계한 패치를 USB로 업로드하면 즉시 실행됩니다. 패치를 바꿀 때마다 완전히 다른 신디사이저가 됩니다.

∞
무한한 사운드 가능성

드럼 머신, FM 신디사이저, 앰비언트 이펙터… 하나의 oscimo가 무수히 다른 악기가 됩니다.

PRESET 01
● SUBTRACTIVE

CLASSIC SYNTH

VCO → VCF → VCA 정통 아날로그 구조. 가장 기본적인 신디사이저 소리를 만듭니다.

  • VCO 피치 제어
  • LOW PASS 필터
  • ADSR 엔벨로프
PRESET 02
● FM SYNTHESIS

FM SYNTH

FM 변조로 금속적이고 복잡한 배음을 가진 소리를 만듭니다.

  • 캐리어 주파수
  • 변조 인덱스
  • 비율 제어
PRESET 03
● PERCUSSION

DRUM ENGINE

킥, 스네어, 하이햇을 oscimo 하나로. 리듬 머신을 직접 설계합니다.

  • 킥 디케이 / 피치
  • 스네어 노이즈
  • 하이햇 필터

🔌
oscimo를 여러 개 연결하면 진짜 모듈러 시스템이 됩니다.
패치 포트를 통해 CV 신호를 주고받으면, 각각의 oscimo가 하나의 모듈이 됩니다. 하나는 VCO로, 하나는 LFO로, 하나는 시퀀서로 — 연결 방식에 따라 시스템이 확장됩니다.
